URL Shortener

  • You can pass an URL as a parameter and you will receive a shortened URL in the JSON response.
  • When you visit that shortened URL, you will be redirectedto the original link.

Example creation usage:

  • hhttps://enormous-panda.glitch.me/new/http://google.com
  • https://enormous-panda.glitch.me/new/http://freecodecamp.com

Example creation output:

{"original_url":"http://freecodecamp.com","short_url":"0"}

Usage:

  • http://eager-chance.glitch.me/2

Will redirect to:

  • http://google.com
